# jwt-persistant

## stacks

- react ui
- go api server

## notice to yale

- hot reload with git bash environment

## getting started

- run server

```
go run .
```

- test with curl

```
curl localhost:3500/ping
```

- run ui

```
cd ui
yarn
yarn run dev
```

- test with ui

```
browser with http://localhost:3500
```
